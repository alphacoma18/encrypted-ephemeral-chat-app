# Encrypted Ephemeral Chat Application

## Tech Stack

- HTML/CSS/JS
- Vite
- Node.js
- Socket.io
- TypeScript

## Frontend

### Installation

```bash
cd front-end

pnpm install
```

### Development

```bash
pnpm dev
```

## Backend

### Installation

```bash
cd back-end

pnpm install
```

### Development

```bash
pnpm start
```
